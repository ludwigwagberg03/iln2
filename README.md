# COOKS

## Beskrivning

COOKS är en webbapplikation utvecklad med React och TypeScript som kombinerar receptinspiration med en enkel köksbutik. Användare kan bläddra bland olika recept eller utforska köksredskap som finns till försäljning. Applikationen design är byggd hjälp av Tailwind CSS och React Router används för smidig navigering mellan sidor.

## Installation

Installera projektets beroenden:

```bash
npm install
npm install tailwindcss @tailwindcss/vite
npm install flowbite
```
